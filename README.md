### How to Contribute

1. Fork the repository
2. Create your feature branch (`git checkout -b feature/NewAgent`)
3. Add your agent to the appropriate platform folder
4. Include documentation and examples
5. Commit your changes (`git commit -m 'Add NewAgent for Platform'`)
6. Push to the branch (`git push origin feature/NewAgent`)
7. Open a Pull Request

- Follow the existing folder structure
- Include a README for new platforms
- Provide clear documentation for each agent
- Test agents thoroughly before submitting
- Use descriptive commit messages
